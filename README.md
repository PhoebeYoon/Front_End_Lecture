 #### 4. 콘솔에서 직접 DOM 노드에 액세스할 수 있는 다양한 방법
엘리먼트 탭에서 하나의 요소를 클릭한 후 콘솔패널로 이동하여 '$0'엔터 합니다. 

$0예를 들어 DOM 트리에서 현재 선택된 노드에 액세스하는 데 사용할 수 있습니다 . 
$1은 크롬 브라우저, $2, $3등을 사용하여 과거 선택의 역순으로 선택된 노드에 액세스할 수 있도록 하여 한 단계씩 거슬러올라가며 보여줍니다.
